In this example, __gcd is a built-in function for calculating the GCD, and the macro gcd(a, b) simplifies its usage. The LCM is calculated using the formula LCM(a, b) = (a * b) / GCD(a, b). The macro lcm(a, b) incorporates this formula.

Remember that using functions instead of macros is often considered better practice in C++, as they provide type safety and better debugging capabilities.
